Frequently Asked Questions

Find answers to common questions about our bot. Whether it's setup, commands, or troubleshooting, we've got you covered!

You can use the /menu command to view the main menu options. This menu allows you to select available features, such as enabling the profanity filter or creating new buttons.

This feature allows the bot to filter messages containing profanity. You can enable or disable this filter by selecting an option from the dropdown provided in the bot's main menu.

Use the /addproduct command with options like product name, price, description, and target channel. The bot will then send an embed with the product information to the specified channel.

You can use the /addmedia command to add a message with media (image/video URL) to the channel. The /addmessage command allows you to send a plain message to the channel.

Use the /chatmember command to send a private message to a specific server member. You can also attach media if needed.

With the /ban command, you can ban a member from the server. Make sure you have the necessary permissions to ban members. You can also provide a reason for the ban.
